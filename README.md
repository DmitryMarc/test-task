## Запуск приложения
Находясь внутри папки с проектом, установить зависимости
```
npm i
```

## Запуск проекта в режиме разработки
```
npm run dev
```
Далее перейти по ссылке [http://localhost:5173/](http://localhost:5173/)

## Сборка проекта
```
npm run build
```
Проект будет собран в папке ./dist
